Sobre mí

Soy una desarrolladora front-end radicado en Perú que busca oportunidades interesantes. Me gusta enfocarme en la accesibilidad al desarrollar. Me apasiona y siento curiosidad por resolver problemas. Actualmente, desarrollo en Angular, React y un poco de diseño. Cuando no programo, disfruto jugando voley y manejando bicicleta. Estoy aprendiendo cada día más para mejorar mis habilidades.

Mis capacidades

Me apasiona el aprendizaje continuo y la evolución constante en el mundo de la tecnología. Como desarrolladora Front-End, me mantengo actualizada en las últimas tendencias, herramientas y frameworks para crear experiencias digitales innovadoras, optimizadas y atractivas. Enfrento cada desafío con determinación, combinando creatividad, rendimiento y usabilidad para desarrollar interfaces que realmente marquen la diferencia.

Mi experiencia

Reestructuré la plataforma de WhiteLion con React v16, actualizándola para una versión más moderna. Lideré el desarrollo de una plataforma APN con Angular, aplicando buenas prácticas de diseño para una interfaz más intuitiva y un código escalable. Implementé mejoras en la escalabilidad del código en diversos proyectos, optimizando la eficiencia del equipo de desarrollo. Además, desarrollé y mantuve interfaces responsivas, colaboré en entornos ágiles con equipos multidisciplinarios y capacité a nuevos desarrolladores para acelerar su integración y productividad.

Desarrollé un módulo en Kotlin para la medición de productividad de desarrolladores, implementando BDD y TDD para mejorar su escalabilidad. Utilicé Cucumber para pruebas unitarias, consumí API Rest y gestioné archivos Excel con la librería POI para generar informes detallados.

Diseñé las interfaces de usuario para las páginas web de Fractio, Costamar SAC, Softnet Innapsis y Fractio, así como las interfaces móviles de Costamar SAC y Fractio. Además, participé en el desarrollo móvil y en proyectos web con Vue. Utilicé tecnologías como HTML, CSS, React Native, Figma, Git, Scrum, Expo Go y Vue.
